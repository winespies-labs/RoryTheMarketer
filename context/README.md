# Context (in-project copy)

This folder is a copy of the workspace **Context** from `WineSpiesCreative/Context`. It lets creative-studio run self-contained without depending on the parent repo layout.

- **Examples/Trustpilot Review Export - Sheet1.csv** — used by the General Ad Builder testimonials dropdown (`/api/reviews`).
- Other files (aboutme, voice guidelines, personas, SOP docs, etc.) are available for future features (prompts, copy tools, etc.).

To refresh from the workspace Context, copy from the repo root:

```bash
cp -r ../../Context/* ./context/
```

(Adjust the path if you're not in `Projects/creative-studio`.)
