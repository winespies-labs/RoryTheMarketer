After moving drafts to the library, what should the editor look like?

The draft list below the editor goes away. What replaces it?

A

Editor is stateless — no list at all

The editor just edits whatever is currently loaded. To switch pieces, you go to the library and click "Open." The editor URL could include the writeup ID (e.g. /copywriting/editor?id=abc). Clean, focused writing environment.

B

Small "current piece" indicator + link to library

A slim bar at the top of the editor shows the current piece title and a "← All writeups" link back to the library. No full list inline — just enough context to know what you're editing.

C

Sidebar list replaces the bottom drawer

Move the writeup list from the bottom drawer to a left sidebar panel within the editor. Still quick to switch without leaving the page, just repositioned.
