Key behaviours

  • Edit prompt expands inline below the card (your choice C) — only one open at a time
  • Save prompt → PUT /api/ad-reference/update → persists to Postgres immediately
  • Add Template card always last — expands a form below the grid
  • Auto-generate from image → sends reference image to Claude Vision → returns a Nano Banana 2 style Gemini prompt pre-filled in the textarea
  • New template requires: image + name + prompt → POST /api/ad-reference/create
  • Select/deselect cards works the same as today — edit state doesn't affect selection
